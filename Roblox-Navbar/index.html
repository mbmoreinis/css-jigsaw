Instructions: We can't get to Roblox in school the usual way, but we can recreate the Roblox Studio navbar menu showed in the image below! It took us a lot of work to set this up and should be pretty easy for you:

  1. Change the list styles to remove the numbers and bullets from thlist items
  2. Give the box class UL a flex display
  3. Remove the underlines from the menu links
  4. Set the hover effect to change the menu item backgrounds
