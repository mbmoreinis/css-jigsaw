Make it a Poster!

CSS Borders on Lists

By Mr. M.

Instructions: This is another version of the Multiplication Table Borders task, but using lists instead of tables.Explore how to style lists with borders and other tweaks to make something worthy of being put on the wall as a poster.

Add these styles in order (see styles in solution.ntml for comments - it's easy):

  1. Center the title "High Factor Times Tables
  2. Display each table in a row with inline-block
  3. Put a rounded 2px black border box around all the tables
  4. Underline the top numbers red
  5. Go night mode on OL LI hovers
  6. Make rounded corner borders on individual tables
  7. Add 20 px margin at bottom of tables
  8. Add a margin at bottom of equation LIs
  9. Add a top margin of 20px on each table
  10. Make sure there's no border or margin on each equation LI
  11. Put 25px padding below each table
