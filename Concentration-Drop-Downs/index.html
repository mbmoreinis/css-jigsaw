Concentration

CSS Dropdowns

By: Mason

Instructions: You'll see a 12-box grid on the Solution Page. Hovering over the first 6 boxes displays overcontent it. With regular drop-down menus, the content is below and shifted, but here, it's directly on top.

Your task:

  1. Add some Dropdown CSS: Dropdowns are based on the :hover pseudoclass, and use the display property to hide the dropdown elements by default. Create a CSS declaration that says that when you hover over the parent, the dropdown-content div should be displayed, and another to give that child element absolute positioning. At that point, it will behave like the video below this list.
  2. Build a Concentration Game board, adding ID selectors to the CSS so that each dropdown 1-6 has a match. Note: two of the boxes have text content in the HTML, so you'll have to add that to the boxes you pick.
  3. Substitute your own images and/or text in place of those here to create your own drop-down concentration game! Based on the sizes of your background images, you may have to adjust things.
  4. Convert Concentration to Flashcards by finding definitions for terms to study. Put the definitions in the HTML dropdown-content divs and the terms in the jargon spans.
