Instructions

Let's look beyond the flat wall of a web page with layered backgrounds! The background-image line determines the source of the background images, while the background-attachment line determines what scrolls or stay fixed.

Start by adding a topographical map background image to the body: it's background/topgraph.png. Give it a fixed background-attachment value so it doesn't move. It should look like the image at left.

Next, you'll add crosshairs background to the div with id="top", Give it an opacity of .1, and position top center. You'll see you can move that around the background top map, as if you're sighting a target.

Last, add a fire-breathing snowman background to id="bottom", which will show up when you click on the crosshairs if you specify the background using the bottom:active pseudoclass.Give that a .1 opacity, a background size of 400px, and a background-clip of content-box so it fits perfectly.

We've included some more fun gifs for you to mess around with and place as you please. Have fun!
